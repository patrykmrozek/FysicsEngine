# FysicsEngine
Lightweight physics engine written in C with an encapsulated API.
Visual examples made with SDL.

## Features (In Development)
- **Rigid Body Dynamics** 
- **Collision Detection**
- **Support for different Shapes**
- **User-friendly API**
